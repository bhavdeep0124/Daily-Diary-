### Summary

Today, I learned about different **Supervised Machine Learning algorithms** used for classification problems. The session began with a brief introduction to **Linear Regression**, followed by **Naïve Bayes**, **K-Nearest Neighbors (KNN)**, and **Support Vector Machine (SVM)**. We understood their working principles, advantages, disadvantages, and real-world applications.

We implemented these algorithms on a **Teen Mental Health Dataset** to predict depression status. The models were evaluated using **Accuracy Score**, **Classification Report**, and **Confusion Matrix**, and we compared their performance to identify the most suitable algorithm for the given dataset.

### Key Learning
- Understanding Naïve Bayes, KNN, and SVM algorithms.
- Training and testing classification models.
- Evaluating models using accuracy, precision, recall, and F1-score.
- Comparing different machine learning algorithms.
- Selecting the best algorithm based on dataset characteristics and performance.
