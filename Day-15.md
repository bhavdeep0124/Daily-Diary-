### Summary

Today, I learned about **Linear Regression** and its regularized versions, **Ridge Regression** and **Lasso Regression**. The trainer explained how Linear Regression predicts continuous numerical values and how Ridge and Lasso Regression help reduce overfitting using regularization techniques. We also learned that Lasso can perform feature selection by removing less important features.

We implemented all three regression models on the **Teen Mental Health Dataset**, evaluated their performance using **Accuracy**, **R² Score**, and **Mean Squared Error (MSE)**, and compared their results to understand the strengths and limitations of each algorithm.

### Key Learning
- Understanding Linear, Ridge, and Lasso Regression.
- Learning the difference between L1 and L2 regularization.
- Reducing overfitting using regularization techniques.
- Performing feature selection with Lasso Regression.
- Evaluating regression models using R² Score and MSE.
- Comparing the performance of different regression algorithms.
