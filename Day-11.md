### Summary

Today, I learned about **Supervised Learning**, a machine learning approach where models are trained using labeled data. The trainer explained the complete workflow, including data collection, feature engineering, train-test splitting, model training, evaluation, and prediction. We also studied the two main types of supervised learning: **Classification**, which predicts categories, and **Regression**, which predicts continuous values.

We implemented a **Logistic Regression** model to predict student placement based on features such as CGPA, attendance, and study hours. We evaluated the model using accuracy and tested it with new student data. Additionally, we applied Logistic Regression to a **Teen Mental Health Dataset** to predict depression status based on factors like social media usage, sleep hours, academic performance, stress level, anxiety level, and addiction level.

### Key Learning
- Understanding the concept of supervised learning.
- Difference between classification and regression.
- Training and testing machine learning models.
- Using Logistic Regression for classification problems.
- Evaluating model accuracy and making predictions on new data.
