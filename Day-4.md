- Multivariate Analysis

### Summary

Today, I learned different techniques for analyzing data before building machine learning models. We studied **Univariate Analysis**, which focuses on analyzing a single variable, and **Multivariate Analysis**, which examines relationships among multiple variables.

The trainer demonstrated the **GroupBy** function in Pandas to summarize data, such as calculating the average CGPA for each department. We also learned about **Crosstab**, which shows the frequency distribution of different categories.

Another important topic was **Correlation Analysis**, which helps measure the strength and direction of the relationship between variables. Finally, we explored **Feature Engineering**, where new features are created from existing data to improve machine learning model performance.

### Key Learning
- Understanding univariate and multivariate analysis.
- Using GroupBy and Crosstab for data analysis.
- Measuring relationships using correlation.
- Creating new features through feature engineering.
- Importance of feature engineering in improving model accuracy.
