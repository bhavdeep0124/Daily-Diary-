### Summary

Today, I learned about **Data Cleaning**, which is the process of improving data quality before analysis. We discussed common data quality issues such as missing values, duplicate records, incorrect data types, inconsistent text formatting, outliers, and extra spaces.

I learned different methods to handle missing values, including removing rows and filling missing values using the **mean, median, and mode**. We also cleaned text data by removing extra spaces and standardizing text formatting.

The trainer explained how to detect outliers using **Box Plots** and the **Interquartile Range (IQR)** method. Finally, we verified the cleaned dataset using Pandas functions like `info()` and `describe()`.

### Key Learning
- Importance of data cleaning in Machine Learning.
- Handling missing values using different techniques.
- Removing duplicate records.
- Cleaning text data.
- Detecting and removing outliers.
- Preparing a clean dataset for analysis and model building.
