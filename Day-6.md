### Summary

Today, I learned about **Feature Engineering**, an important step in Machine Learning that involves transforming raw data into meaningful features to improve model performance. We studied different feature engineering techniques such as creating new features, transforming existing features, scaling numerical data, encoding categorical data, and binning continuous values into categories.

We created new features like **Performance Score**, **Backlog Ratio**, and **Study Efficiency** from existing columns. We also learned feature transformation using logarithmic transformation, applied **StandardScaler** and **MinMaxScaler** for feature scaling, used **One-Hot Encoding** and **Label Encoding** for categorical data, and grouped attendance values into categories using feature binning.

### Key Learning
- Importance of feature engineering in Machine Learning.
- Creating new features from existing data.
- Transforming features to improve data distribution.
- Scaling data using StandardScaler and MinMaxScaler.
- Encoding categorical variables into numerical values.
- Using feature binning to categorize continuous data.
